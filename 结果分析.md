

  整体轨迹与指标

  - traj_result.png
      - 来源：plot_trajectories（plot.py）
      - 内容：在 x–y 平面上画出 A/B 的真值轨迹（实线）和 GLMB 估计轨迹
        （点/虚线）。
      - 用途：一眼看整体跟踪质量、交汇段是否跟上、ID 有无明显漂移。
  - metrics_over_time.png
      - 来源：plot_metrics_over_time（plot.py）
      - 结构：3 行子图，共享时间轴。
          - 上：OSPA 随时间变化（单位 m）。
          - 中：位置 RMSE 随时间变化（单位 m）。
          - 下：目标数（Cardinality）真值 vs 估计（虚线 = Truth，实线
            = Est）。
      - 用途：量化整体精度、目标数误差在时间上的变化。
  - metrics_summary.txt
      - 来源：main.py 写入。
      - 内容：RMSE、平均 OSPA、基数误差、混淆矩阵数字、假目标总数和干扰区
        内假警次数等数值摘要。
      - 用途：论文/PPT 里的具体数字引用。

  检测率 / ID / 混淆

  - detection_recall.png
      - 来源：plot_detection_recall（plot.py）
      - 内容：A 与 B 的“累积召回率”曲线（横轴时间，纵轴 0–1）。
      - 用途：看跟踪器对 A/B 的长期检测率是否足够高，有无明显掉线时段。
  - confusion_matrix.png
      - 来源：plot_confusion_matrix（plot.py）
      - 结构：2×3 热力图，
          - 行：True A / True B
          - 列：Pred A / Pred B / Pred None（漏检）
      - 用途：
          - A→B、B→A 的 ID 混淆（off-diagonal）；
          - None 列看 A/B 漏检帧数；
          - 结合 metrics_summary.txt 里面的 A_as_B, B_as_A, A_missed,
            B_missed。
  - id_timeline.png
      - 来源：plot_id_timeline（plot.py）
      - 内容：
          - 横轴：时间；
          - 纵轴：滤波器内部的 label ID（0,1,2,…）；
          - 点的颜色：当前这一 label 对应的真值目标 A/B/None。
      - 用途：
          - 看每条 label 在时间上是否始终对应同一个真值（ID 一致性）；
          - 交汇/干扰段是否发生 ID-switch（颜色突变）。

  量测层面的可视化

  - meas_step60.png
      - 来源：plot_measurements_snapshot（plot.py，默认取第 60 帧）。
      - 内容：
          - 当前帧所有雷达的量测点云：
              - 不同雷达 → 不同 marker；
              - is_clutter=False 真目标点和 is_clutter=True 杂波点分开画；
          - 可选叠加：当前帧的 A/B 真值位置和估计位置。
      - 用途：
          - 看某一帧的杂波分布、扩展目标点云形状、真值和估计的位置关系。
          - 需要其他帧时可在 main.py 里改 snapshot_idx。
  - radar_detection_over_time.png
      - 来源：plot_radar_detection_over_time（plot.py）
      - 结构：3 行子图，分别是 R1 / R2 / R3。
          - 横轴：时间；
          - 纵轴：该雷达在每一帧的“有效目标量测数”（仅统计 is_clutter ==
            False 的 A/B 量测）。
          - R2 子图里额外有 [JAM_START_T, JAM_END_T] 的红色阴影（压制干扰
            时间窗）。
      - 用途：
          - 直观看每个雷达对 A/B 的“看到多少点”；
          - 比较 jamming 前/中/后的 R2 曲线是否在阴影段接近 0，同时对比
            R1/R3 是否正常；
          - 结合距离图判断点目标区（每帧≈1点）和扩展目标区（每帧≈10点）。
  - radar_distance_over_time.png
      - 来源：plot_radar_distance_over_time（plot.py）
      - 结构：3 行子图（R1/R2/R3），每个子图：
          - 横轴：时间；
          - 纵轴：该雷达到最近目标（A 或 B）的距离（km）；
          - 黑色虚线：50 km 阈值。
      - 用途：
          - 辅助理解点目标 vs 扩展目标区：
              - 距离 > 50 km → 对应点目标量测（每帧 1 个）
              - 距离 < 50 km → 对应扩展目标量测（每帧 8–15 个）
          - 可以和 radar_detection_over_time.png 对照，看“距离穿过 50km
            时，量测数从 1 跳到十几个”。

  - result.gif
      - 来源：dynamic_sim.py 里保存的动画（你运行动态仿真时生成）。
